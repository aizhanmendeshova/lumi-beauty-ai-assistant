# LUMI Beauty AI Assistant ✨💅

## Overview
**LUMI Beauty AI Assistant** is an enterprise-grade, conversational AI application designed specifically for a premium beauty studio. Powered by Google's Gemini LLM (1.5 Flash) and built with Python and Streamlit, LUMI automates client interactions, provides instant pricing information, and manages booking inquiries. 

The architecture strictly adheres to modern enterprise standards, implementing structured JSON outputs, hallucination prevention mechanisms, and built-in Operations Management (FinOps & D2C Analytics).

---

## 🛠 Tech Stack
- **Language:** Python 3.10+
- **Frontend Framework:** Streamlit (Custom CSS styling, stateful memory)
- **AI/LLM Engine:** Google GenAI SDK (`google-genai>=1.0.0`) using the Gemini 1.5 Flash model
- **Data Validation:** Pydantic (Strict schema enforcement)
- **Environment Management:** `python-dotenv`

---

## 🏗 Project Architecture & Core Modules

The project is broken down into highly cohesive, modular components:

### 1. `config.py` (Configuration & Strategy)
Acts as the brain's configuration center. It securely loads environment variables and stores globally shared configurations:
- **FinOps Data:** Defines the baseline cost per 1 Million input/output tokens to calculate real-time studio expenses.
- **Business Logic:** Contains the hardcoded, up-to-date service price list (e.g., Manicure, Pedicure, Nail Extensions) featuring both fixed prices and dynamic ranges.
- **Prompt Engineering:** Houses the `SYSTEM_PROMPT` which instructs the AI on its personality, decision-making protocols, and strict JSON output formatting.

### 2. `beauty_models.py` (Data Schema & Validation)
Leverages `Pydantic` to guard against AI hallucinations.
- Defines `LumiResponseSchema`, ensuring the LLM always returns a predictable, strongly-typed JSON object.
- **Fields include:** `intent`, `service`, `cost_kzt`, `message_to_client`, `confidence_score`, and `is_ready_for_human`.
- If the model attempts to output invalid data, Pydantic's `ValidationError` isolates the fault, protecting the application layer.

### 3. `lumi_ops_manager.py` (FinOps & D2C Analytics)
A backend utility responsible for operational audits and logging (`lumi_business.log`).
- **FinOps (Financial Operations):** Calculates the exact monetary cost of every single query by analyzing the Prompt and Completion token headers natively returned by Gemini. Converts USD costs to local runtime currency (KZT).
- **D2C (Direct-to-Consumer Quality):** Evaluates the `confidence_score` generated by the model. Flags outputs as `SAFE` or `REVIEW_REQUIRED` (if confidence drops below 70%), empowering human managers to step in seamlessly.

### 4. `lumi_ai_assistant.py` (The Interactive UI Layer)
The user-facing Streamlit application.
- **Aesthetics:** Implements a premium, custom CSS layout featuring soft pink/cream gradients and iOS-style rounded chat bubbles.
- **Contextual Memory:** Utilizes `st.session_state` to maintain an active conversation history, passing historical context natively into the GenAI `contents` payload.
- **Real-Time Dashboard:** A responsive sidebar that displays:
  - The live FinOps monetary session cost.
  - A dynamic progress bar representing the AI's real-time confidence score.
  - A beautifully formatted menu of studio services dynamically loaded from the config.

## 🚀 Setup & Installation Guide

### Prerequisites
Make sure you have Python installed on your machine.

### 1. Clone or Extract the Repository
Navigate to the central project folder (`beauty_studio_ai` directory).

### 2. Install Dependencies
Run the following command to install the required libraries:
```bash
pip install -r requirements.txt
```

### 3. Environment Setup
Create a `.env` file in the root directory and add your Google Gemini API Key:
```env
GEMINI_API_KEY=your_google_api_key_here
```

### 4. Launch the Application
Start the Streamlit development server:
```bash
streamlit run lumi_ai_assistant.py
```
A browser window will automatically open with the LUMI Assistant GUI.

## 📊 Logging and Auditing
Every conversation interaction is securely logged inside the auto-generated `lumi_business.log` file.
**Sample Log Entry:**
```text
[2026-04-14 18:20:05] INTENT: price_check  | TOKENS: 450 in / 112 out | COST: 0.0385 ₸ (0.000077 $) | CONFIDENCE: 0.95 -> [SAFE]
```
This allows studio owners to monitor AI token expenditure and evaluate the bot's accuracy simultaneously.
